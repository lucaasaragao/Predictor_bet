Radar de Palpites

Política de Privacidade

Última atualização: 20 de abril de 2026

1. Comprometimento com sua Privacidade

O Radar de Palpites foi criado com foco em privacidade. Nesta página, explicamos de forma clara quais dados são usados, como e por quê.

2. Dados Coletados

Este site não coleta, armazena nem envia dados pessoais dos usuários. Em resumo:

  • Não há cadastro nem login
  • Não usamos cookie de rastreamento nem analytics
  • Nenhum dado é enviado para servidores de terceiros

3. Armazenamento Local (sessionStorage)

Para controlar o acesso ao conteúdo, usamos o sessionStorage do navegador. Essa tecnologia:

  • Guarda apenas a confirmação de que o usuário é maior de 18 anos
  • Existe somente enquanto a aba do navegador está aberta
  • É apagada automaticamente ao fechar a aba ou o navegador
  • Nunca é enviada para servidor externo
✓ Seus dados não saem do seu dispositivo.

4. Dados de Partidas e Estatísticas

As informações das partidas usadas nas análises vêm da API pública football-data.org e são processadas localmente pelo script predictor.py. Esses dados são de jogos e times, não de usuários.

5. Links Externos

Este site pode ter links para redes sociais (Instagram e LinkedIn). Ao abrir esses links, passam a valer as políticas de privacidade de cada plataforma.

6. Alterações nesta Política

Esta política pode ser atualizada conforme o serviço evoluir. A data no topo da página sempre mostra a versão mais recente.

7. Contato

Se tiver dúvidas sobre privacidade, fale comigo pelo LinkedIn ou Instagram.
